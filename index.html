📖 Manuel SIGLOF v5

Tableau de bord

Filtrage actif sur la période, le groupe NSC, l'unité et le statut. Les indicateurs (disponibilité, ruptures, alertes, mouvements) se recalculent à chaque changement de filtre.

Gestion des stocks — G10 Armes

Filtres : Classe NSC, Unité, Statut, Type, Modèle, Calibre. Colonnes Type / Sous-type / Modèle / Calibre s'affichent uniquement quand G10 est sélectionné.

Gestion des stocks — G13 Munitions

Filtres exhaustifs : Classe, Calibre, Type de balle, Type de caisse, Origine. Colonnes : Type, Calibre, Type balle, Type caisse, Origine, Nb caisses, Boîtes/caisse, Cart./boîte, Total/caisse, Total cartouches.

Mouvements — G10 Armes

Le formulaire affiche les champs Classe NSC, Code Suivi/NS-BF, Désignation, Type, Sous-Type, Modèle, Calibre avec listes dynamiques selon la classe. Le champ Quantité est supprimé — la traçabilité se fait par Code Suivi individuel.

Mouvements — G13 Munitions

Le formulaire de mouvement est identique au formulaire d'enregistrement SSAM-BF : Classe NSC en premier, formulaire reconfigurable selon la classe (format standard, obus, mines, bombes, vrac, missiles, détonateurs).

Dotations unités

Filtrage par groupe NSC et unité. Taux de dotation par catégorie (Armement, Protection, Véhicules, Vivres, Médical, Carburants).

Rapports

Génération de rapports PDF : inventaire mensuel, dotations, mouvements, alertes de stock, rapports ministériels.
